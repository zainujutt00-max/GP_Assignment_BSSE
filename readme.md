# Assignment: Multiplayer Game Code Architecture Design


## Task

Choose any modern multiplayer online game that involves **more than two players**. Examples include:

* Battle Royale games
* MOBA games
* Team-based shooters
* Multiplayer survival games
* MMORPGs
* Any other multiplayer game of your choice

Some of the options include War Thunder, Dota 2, CS2, League of Legends, Apex Legends, Among Us etc.

Once you have selected a game, prepare a **PDF document** describing how you would design the code architecture for that game.

Your document should include:

### 1. Game Overview

A brief description of the selected game and its core gameplay mechanics.

### 2. High-Level Architecture

Explain the major systems required to build the game. For example:

* Player System
* Matchmaking System
* Combat System
* Inventory System
* Game State Management
* UI System

### 3. Code Structure

Describe how you would organize the codebase:

* Classes
* Components
* Responsibilities of each system
* Communication between systems

Diagrams are highly encouraged.

### 4. Design Decisions

Explain why you chose this architecture and why you believe it is the best approach for the selected game.

The focus of this assignment is on **software architecture, and technical reasoning**, not on game design details.
